**Root Cause**: The `mng_order.php?order_day=25323` page contains data for only ONE team (the team that owns that page), but the system was using the same data for both teams.

### Solution Implemented

#### 1. Added Team Detection Function
```javascript
function detectTeamFromHTML(htmlText) {
    // Extracts team ID from JavaScript variable 'curr' in HTML
    const currMatch = htmlText.match(/var\s+curr\s*=\s*(\d+)/);
    if (currMatch) {
        const teamId = parseInt(currMatch[1]);
        return teamId;
    }
    return null;
}
```
