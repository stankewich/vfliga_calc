### Особенности дизайна
1. **Средний вариант выделен** - голубой фон колонки
2. **Цветовое кодирование**:
   - Хозяева: зеленый
   - Гости: красный
   - Разница: синий
3. **Иконки погоды** - визуальная индикация
4. **Моноширинный шрифт** для чисел - легче сравнивать
5. **Hover эффект** - подсветка строки при наведении

### Адаптивность
Для узких экранов (<600px) можно переключиться на вертикальный вариант через media query.

## Альтернатива: Сворачиваемая таблица

Если места мало, можно сделать:
1. По умолчанию показывать только средний вариант
2. Кнопка "Показать все варианты" раскрывает полную таблицу
3. Сохранять выбор пользователя в localStorage

```
┌─────────────────────────────────────────┐
│ Сила хозяев: 1245.78                    │
│ Сила гостей: 995.43                     │
│ Разница: +250.35                        │
│                                         │

└─────────────────────────────────────────┘
```
